# Hotel Booking Data Analysis Dashboard

This project analyzes hotel booking data using Microsoft Excel to understand booking patterns, cancellations, and customer behavior.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization

## Analysis Performed
- Booking trends by month
- Cancellation analysis
- Resort vs City hotel comparison
- Customer stay duration

## Key Insights
* City hotels received more bookings than resort hotels.
* Cancellation rate is higher in city hotels.
* Most bookings occur during peak travel months.
* Resort hotels have longer average stay durations.
